## BIENVENIDA

Hola, bienvenido o bienvenida al **Curso Completo de Notion en Español de Notion LATAM**.

Este workbook es tu compañero de aprendizaje durante todo el curso. Está diseñado para que no solo veas los videos, sino que pongas en práctica cada concepto de forma inmediata.

**Cómo usar este workbook:**

1. **Ve primero el video** de cada lección antes de leer la sección correspondiente
2. **Completa los ejercicios** antes de avanzar a la siguiente lección
3. **Anota tus reflexiones** en los espacios en blanco — el aprendizaje real ocurre cuando aplicas los conceptos a tu contexto
4. **Construye los proyectos** de cada módulo con información real de tu trabajo
5. **Usa el glosario** cada vez que encuentres un término que no conoces

## GLOSARIO DE TÉRMINOS NOTION (30 términos)

| Término | Definición |
|---------|-----------|
| **All-in-one workspace** | Plataforma que integra múltiples herramientas de trabajo en un solo lugar |
| **Block** | La unidad básica de contenido en Notion. Todo es un bloque (párrafo, imagen, base de datos, etc.) |
| **Board view** | Vista de tablero Kanban que agrupa registros por columnas de estado |
| **Breadcrumb** | Navegación jerárquica que muestra la ruta de páginas desde la página actual hasta la raíz |
| **Calendar view** | Vista de calendario que muestra registros según sus fechas |
| **Callout block** | Bloque destacado con ícono y fondo de color para resaltar información |
| **Cover** | Imagen de portada de una página |
| **Database** | Colección de registros estructurados con propiedades, vistas y filtros |
| **Filter** | Condición para mostrar solo los registros que cumplan ciertos criterios |
| **Formula** | Propiedad calculada basada en otras propiedades usando expresiones |
| **Full page database** | Base de datos que ocupa toda la pantalla como una página independiente |
| **Gallery view** | Vista de tarjetas visuales que muestra la imagen de cada registro |
| **Icon** | Emoji o imagen que identifica visualmente una página |
| **Inline database** | Base de datos embebida dentro de una página junto a otro contenido |
| **Linked database** | Vista de una base de datos existente insertada en otra página |
| **Multi-select** | Propiedad que permite seleccionar múltiples valores predefinidos |
| **Nested page** | Página que vive dentro de otra página en la jerarquía |
| **Notion AI** | Funcionalidad de inteligencia artificial integrada en Notion para asistir con escritura y análisis |
| **Property** | Campo de datos de una base de datos (columna) con un tipo específico |
| **Relation** | Propiedad que conecta dos bases de datos diferentes |
| **Rollup** | Propiedad que agrega datos desde una base de datos relacionada |
| **Select** | Propiedad que permite elegir un solo valor de una lista predefinida de opciones |
| **Sidebar** | Barra lateral de navegación que muestra todas las páginas del workspace |
| **Slash command** | Menú de comandos que se activa escribiendo "/" para insertar bloques |
| **Sort** | Ordenamiento de registros en una base de datos por una o más propiedades |
| **Status** | Propiedad especial para gestión de proyectos con estados predefinidos |
| **Template** | Modelo preconfigurado de página o registro que puede reutilizarse |
| **Timeline view** | Vista de diagrama de Gantt para visualizar proyectos con duración |
| **Toggle** | Bloque desplegable que oculta/muestra contenido al hacer clic |
| **Workspace** | El espacio de trabajo principal que contiene todas las páginas y bases de datos |
